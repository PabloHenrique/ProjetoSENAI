Bem vindo!

Este projeto foi criado buscando facilitar a aplicação dos conceitos eletroeletrônicos estudados na escola SENAI - Marília SP!

Iniciamente, este site foi pensado para ser somente um conversor das grandezas elétricas, porém agora, ele é focado em agilizar qualquer conhecimento elétrico, como o calculo de resitências, fórmulas, resistividade... entre outros.
